## [场景1：事件——冷开场]
[地点：拥挤的夜市，霓虹灯色彩渗透进雨水打湿的地面。七年前。]
[时间：傍晚，下雨]
[背景音乐：不谐和的合成器。沉闷的人群噪音。]

[显示 星河 - 年轻的，19岁，表情震惊]

*七年前。那时我不知道自己是什么。*

*我只知道我无法呼吸。*

[人群压来。到处都是身体。千万个人的情感残留——兴奋、渴望、疲惫、欲望、焦虑——如海啸般冲击着星河未加防护的共鸣，势不可挡。]

[音效：远处的坠落声。尖叫。]

人群中有人倒下了。一个女人。她在抽搐，眼睛翻白。

医护人员
"让开！全部让开！叫救护车！"

[星河想退开，但被困住了，周围都是人。情感的混乱加剧。恐惧。惊慌。他能感觉到每一个人的恐惧，就像是他自己的。他的呼吸断断续续，急促而浅促。]

*我能感觉到他们。所有人。我不理解。现在也不。*

[音效：声音消褪。隧道效应。星河的视野收缩。]

[CG：雨水浸湿的市场旋转。霓虹灯分裂成白噪音。]

*画面淡黑。*

*我花了七年的时间试图忘记这一刻。我很擅长遗忘。*

## [场景2：七年后——日常]
[地点：星河的公寓——小，简约，极度整洁]
[时间：清晨，窗户被雨水模糊的灰蒙蒙光线]
[背景音乐：安静的环境音。外面的都市生活——远处的车流，施工]

[显示 星河 - 26岁，表情疲惫但锐利。黑眼圈是他脸上的常客。]

[闹钟：6:47 AM。星河的手在第三声鸣叫前移动。高效。习惯成自然。]

旁白（内心）
*咖啡。洗澡。黑色衬衫，黑色长裤。不会暴露的内衣。一切都选得这么仔细，仔细到好像没有选择一样。*

[显示星河在厨房里冲咖啡。公寓很稀疏——书，一张薄床垫，没有多余的东西。这是一个不想留下痕迹的人的空间。]

旁白（内心）
*七年的同样日常。人们说习惯可以适应任何东西。压抑也不例外。只要你足够坚持。*

[星河的手机嗡嗡振动。他扫了一眼，没有触碰。"京浩"这个名字短暂地显示，然后屏幕黑了。]

旁白（内心）
*办公室的京浩。大概在问我是否会参加组队午餐。我不会。我从不会参加组队午餐。*

[显示星河的表情：冷漠，疲惫]

## [场景3：办公室——咨询公司]
[地点：玻璃钢铁高层。星河的办公桌在角落，背对着主办公室]
[时间：傍晚，灰色的天空透过落地窗显现]
[背景音乐：中立的公司环境音。键盘点击声。]

[显示 星河 - 表情注释为"烦恼"]

京浩
（走近，过于热情）
"星河！来吧。大家都要去楼下面条店吃饭。你总得吃东西吧。"

星河
"我有工作要做。"

京浩
"你总是在工作。你需要社交。我好像从来没有看过你真正微笑过。"

[星河在键盘上的手指停顿。他转身刚好足以迎上京浩的目光，而在他这样做的时候，他的表情柔和了大约0.3度。]

星河
"我笑过一次。在2019年。那不太合适。"

[选择]
> "你需要多出去。来吧，我请客。"（京浩坚持，带着真挚的温暖）
> "好吧，只是...你知道我很奇怪，对吧？"（京浩带着笑退缩）

[无论选择哪一个，星河都没有去吃午饭。京浩离开后，他坐在安静的办公室里，透过玻璃墙看着他们。他们在办公桌边聚集在一起的方式。他们玩笑的轻松。他们毫不自觉地相互触碰对方手臂的方式。]

旁白（内心）
*我很擅长读人。即使不使用共鸣，我也能看到断层线。京浩笑是为了隐藏他很孤独。会计部的朴志因为妻子出轨所以喝得太多。实习生民成，她焦虑的时候会摸她的项链，因为她去世的母亲给了她这条。*

*我知道他们所有的秘密。情感残留不会褪去。它只是坐在那里，等着某个足够不谨慎的人去倾听。*

*我不再那么不谨慎了。*

[显示星河转身回到电脑，在空荡荡的办公室里孤立，而他的同事们远处的笑声透过玻璃回荡。]

## [场景4：酒吧——相遇]
[地点：小型地下酒吧。昏暗的灯光，昂贵的威士忌，那种人们来独自相聚的地方]
[时间：下班后的晚上。窗外天已黑]
[背景音乐：爵士乐。有点忧伤。钢琴和贝斯。]

[显示 星河 - 表情疲惫，喝纯威士忌，不享受]

*我有时来这里。不是为了威士忌。为了安静。大多数酒吧太吵——太多人，太多情感压在墙上。这个地方不同。酒保把音乐放得足够响，淹没一切。*

[一个男人坐在星河旁边的凳子上。这个男人立刻就很引人注目：银白色的头发垂入眼睛，昂贵的衣服穿得就像他没有想过它们，一个如相机闪光般出现又消失的微笑。]

[显示 凯 - 表情迷人。他的绿眼睛在昏暗的酒吧灯光中不可思议地明亮。]

[这是凯。星河现在还不知道。但他会的。]

凯
"你看起来好像在结束什么东西。一段感情？一份工作？一段真正重要的友谊？"

星河
（不看他）
"都不是。"

凯
"那就是你自己。那更糟。"

[星河终于转身。缓缓地。就像有人在转身看一只危险的动物闯入他的空间。]

星河
"你总是这样骚扰酒吧里的陌生人，还是我特别？"

凯
（微微靠近，微笑扩大）
"你是我见过最糟糕的骗子。你的共鸣在尖叫，即使你把它锁得这么紧，你已经忘记了自己的感觉是什么样的。"

[停顿。完全的沉默。星河的世界裂开了。]

[显示 星河 - 表情震惊，然后愤怒]

星河
（声音低沉，危险）
"离我远点。"

凯
"做不到。你正是我一直在找的人。"

[星河站起来。凯没有动——只是带着那令人恼火的微笑看着他。]

星河
"我不知道你在说什么。我不认识你。我也不想认识。"

[他留下钱在吧台上，开始走向出口。]

凯
（大声喊在他身后）
"我叫凯。你即将发现忽视这个会比逃离它困难得多。"

[显示 星河 - 表情愤怒，手轻微颤抖地推门]

[选择]
> 回来质问凯知道什么（对抗路线）
> 继续走，试图假装这从未发生过（回避路线）

[如果对抗：星河转身，重新进入酒吧，坐回吧台凳子。]

星河
"开始说话。"

凯
"不在这里。不是这样。你距离惊恐发作大概还有三十秒钟，如果你在酒保面前发作那就太不幸了。"

[如果回避：星河在雨中走回家，反复重放对话，他的压抑七年来第一次动摇。]

旁白（内心）
*不知道他知道什么比任何他能给我的回答都更糟。*

## 制作说明

### 角色表情图

**星河（8个关键表情）**
1. **中立** - 默认；警惕，只有那些知道如何看的人才能读取
2. **疲惫** - 压抑的疲惫；悲伤的眼睛，紧绷的下巴
3. **烦恼** - 干燥，切割，冷漠的面具
4. **惊讶** - 稀有；当某些东西真正突破
5. **脆弱** - 面具掉落；底层的真实的人
6. **愤怒** - 强烈，受控的愤怒；最危险的冷
7. **微笑稀有** - 真实的微笑，为真实连接的时刻保留
8. **共鸣激活** - 眼睛微微发光；整个举止转变为直觉，超意识

**凯（8个关键表情）**
1. **迷人** - 实践的面具；千瓦特微笑，似乎轻松
2. **调皮** - 面具的调皮版本；真实的幽默分层下面
3. **认真** - 面具完全脱落；尖锐的智力可见
4. **受伤** - 脆弱；外观开裂
5. **面具脱落的脆弱** - 完全无防护；美丽和破碎
6. **调情** - 真实版本，不实践的版本；真实欲望
7. **强烈** - 专注，存在，所有防御都消失了
8. **共鸣激活** - 眼睛以琥珀色金色光发光；变得超敏感，几乎狂热的力量

### CG时刻（4总）

1. **场景1** - 星河在市场的第一个共鸣爆发（简短，没有明确的CG——隐含）
2. **场景7** - 第一次共鸣会议（两个身体在蓝色和金色光中缠绕）

### 音频设计
- 背景音乐应该从不谐和和不安（场景1-3）转变为亲密和建筑（场景7-11）到背叛和恐惧（场景12）
